# ğŸš‘ Aarambh 112 - Next-Gen Emergency Response System

**Aarambh 112** is an AI-powered Computer Aided Dispatch (CAD) system designed to streamline emergency response operations.  
It unifies ultra-low latency **Voice AI**, a **Real-Time Dispatch Dashboard**, and **RAG-based Protocol Assistance** into a single cohesive platform.

The system automates emergency call intake, extracts critical incident data (Location, Casualty Count, Danger Level), generates structured reports, and guides dispatchers through Standard Operating Procedures (SOPs).
